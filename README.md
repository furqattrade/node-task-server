
## Run Project Locally

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd node-task-server
```

Install dependencies

```bash
  yarn install
```
Running project in dev mode

```bash
yarn watch
```
Running project in dev mode

```bash
yarn start
```


## Features

- User registration
- CRUD operation of Task
